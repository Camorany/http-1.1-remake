# HTTP/1.1 Protocol Remake

### A learning-focused project that explores the inner workings of the HTTP/1.1 protocol by implementing it from the ground up. The goal of this project is to gain hands-on experience with web communication, request/response handling, and data streaming at the protocol level.

## Features

* **Chunked Transfer Encoding** – Understanding how large or dynamically generated content can be sent in pieces without knowing the total size beforehand. <br><br>
* **Video Content Support** – Serving video files with proper MIME types to learn how HTTP handles different content types. <br><br>
* **Request and Response Handling** – Building familiarity with HTTP methods, headers, and status codes through practical implementation. <br><br>
